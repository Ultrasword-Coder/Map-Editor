# Map Editor

## How to Set-Up

- Just do it :D
- I plan on releasing an exe so things should be fine

## Controls

- Toggle tile visibility:
  LCONTROL + LSHIFT + T

- Toggle entity visibility
  LCONTROL + LSHIFT + E

- Change Art type (tile art)
  LCONTROL + T

- Change Art type (entity art)
  LCONTROL + E
